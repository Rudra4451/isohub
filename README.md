ISO Hub

ISO Hub is a lightweight website that helps users find official operating system ISO downloads and learn how to install them.
Instead of hosting files, the site redirects users to the official sources such as Microsoft, Canonical, or Fedora.

---

🚀 Features

- 📥 Quick access to official OS download pages
- 🧭 Step-by-step OS installation guide
- 📊 Operating system comparison table
- 🐧 Directory of popular Linux distributions
- ❓ FAQ section for common questions
- 🔎 Search and filter for operating systems
- 🌙 Modern dark UI design

📖 How It Works

1. User selects an operating system.
2. The site opens a page with system requirements and installation steps.
3. Download buttons redirect to the official OS website.

ISO Hub does not host any ISO files.

---

⚡ Installation / Run Locally

Clone the repository:

Open the project folder and launch:
