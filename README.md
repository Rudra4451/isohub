⚙ How It Works

1. User selects an operating system.
2. The page shows system requirements.
3. Installation steps are displayed.
4. Download button redirects to the official website.

⚠ ISO Hub does not host any ISO files.

---

🚀 Run Locally

Clone the repository:

git clone https://github.com/rudra4451/isohub.git

Open the folder and launch:

index.html
