Step-by-Step

How to Change Your OS

Follow these steps to safely install any operating system on your laptop or desktop.

01
⬇️
Download the ISO
Click any button above. You'll be taken to the official site. Download the ISO file (usually 1–5 GB) and save it somewhere easy to find.
02
💾
Create a Bootable USB
Use Rufus (Windows) or balenaEtcher (Mac/Linux) to write the ISO to a USB drive (8 GB+). This makes the USB into an installer.
03
💽
Backup Your Files First
Copy important files to an external drive or cloud storage. A clean install will erase the target drive — don't skip this step!
04
🔧
Enter Boot Menu
Restart your PC and press F2, F12, DEL, or ESC (depends on brand) to open the boot menu. Select your USB drive to boot from it.
05
⚡
Run the Installer
Follow the on-screen wizard. Pick language, keyboard, and where to install. Choose "Erase disk and install" for a clean fresh start.
06
🎉
Boot Your New OS
Installation takes 10–30 minutes. Remove the USB when prompted, restart — and welcome to your brand new operating system!
