How to Install Ubuntu

  • 1
    Download the Ubuntu Desktop ISO from the official link above. About 5–6 GB.
  • 2
    Download balenaEtcher (free) and flash the ISO to your USB drive (8 GB+).
  • 3
    Back up important files. Restart and press F12/F2/DEL to enter the boot menu.
  • 4
    Ubuntu loads a live preview — try it before installing, no changes made yet!
  • 5
    Click "Install Ubuntu". Choose "Erase disk and install Ubuntu" for a clean install.
  • 6
    Set username and password. Takes 10–20 minutes. Restart and remove USB.
Tip: Ubuntu's live mode lets you try the full OS without installing anything. Click "Try Ubuntu" on the first screen!
