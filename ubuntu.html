How to Install Ubuntu

  • 1
    Download the Ubuntu Desktop ISO from the official link above. About 5–6 GB.
  • 2
    Download balenaEtcher (free, from etcher.balena.io) and use it to flash the ISO to your USB drive.
  • 3
    Back up important files. Plug in USB, restart and enter the boot menu (F12, F2, or DEL).
  • 4
    Select your USB. Ubuntu loads a live preview — you can try it before installing!
  • 5
    Click "Install Ubuntu" and select "Erase disk and install Ubuntu" for a clean install.
  • 6
    Set username and password. Installation takes 10–20 minutes. Restart when prompted.
Tip: Ubuntu's live mode lets you try the OS without installing anything. Click "Try Ubuntu" on the first screen to test before committing.
