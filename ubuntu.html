How to Install Ubuntu

  • 1
    Download the Ubuntu Desktop ISO from the official link above. It's about 5–6 GB.
  • 2
    Download balenaEtcher (free, from etcher.balena.io) and use it to flash the ISO to your USB drive.
  • 3
    Back up important files. Plug in your bootable USB, restart and enter the boot menu (F12, F2, or DEL).
  • 4
    Select your USB from the boot menu. Ubuntu will load a live preview — you can try it before installing!
  • 5
    Click "Install Ubuntu", choose your language, and select "Erase disk and install Ubuntu" for a clean install.
  • 6
    Set your username and password. Installation takes 10–20 minutes. Restart when prompted and remove USB.
💡 Tip: Ubuntu's live mode lets you try the OS without installing anything. Click "Try Ubuntu" on the first screen to test it out before committing to install it.
