How to Install Linux Mint

  • 1
    Download the Cinnamon ISO from the Linux Mint site above. About 2.8 GB.
  • 2
    Use Rufus or balenaEtcher to flash the ISO to your USB drive (8 GB+).
  • 3
    Back up your data then restart and press F12/F2/DEL for the boot menu.
  • 4
    Mint boots into a live desktop — try it out before deciding to install!
  • 5
    Double-click "Install Linux Mint". Choose "Erase disk" for a clean install.
  • 6
    Set username and password. Finishes in 10–20 minutes. Remove USB and restart.
Tip: The live session lets you browse the internet and test software before installing anything!
