How to Install Linux Mint

  • 1
    Download the Cinnamon ISO from Linux Mint's site above. About 2.8 GB.
  • 2
    Use Rufus (Windows) or balenaEtcher to flash the ISO to your USB drive (8 GB+).
  • 3
    Back up your data then restart and press F12/F2/DEL to open the boot menu.
  • 4
    Select the USB drive. Linux Mint boots into a live desktop — try it before installing.
  • 5
    Double-click "Install Linux Mint" on the desktop. Choose "Erase disk" for a clean install.
  • 6
    Set up username and password. Installation finishes in 10–20 minutes. Remove USB and restart.
