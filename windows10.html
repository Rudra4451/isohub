How to Install Windows 10

  • 1
    Download the Media Creation Tool or ISO from Microsoft's site above.
  • 2
    Insert a USB drive (8 GB+) and flash the installer onto it using the tool or Rufus.
  • 3
    Back up all your files to cloud or external storage first.
  • 4
    Restart and press F2, F12, DEL, or ESC to enter BIOS. Set USB as first boot device.
  • 5
    Select "Custom: Install Windows only" and follow the setup wizard.
  • 6
    Takes 20–40 minutes. PC restarts several times — this is completely normal!
