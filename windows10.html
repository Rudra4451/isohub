How to Install Windows 10

  • 1
    Download the Media Creation Tool or ISO file from Microsoft's site (link above).
  • 2
    Insert a USB drive (8 GB+) and use the tool or Rufus to flash the installer onto it.
  • 3
    Back up all your files to cloud or external storage before proceeding.
  • 4
    Restart your PC and press F2, F12, DEL, or ESC to enter the BIOS. Set USB as the first boot device.
  • 5
    The Windows installer will load. Select "Custom: Install Windows only" for a clean install.
  • 6
    Installation takes 20–40 minutes. Your PC will restart several times — this is completely normal.
