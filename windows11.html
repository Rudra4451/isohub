How to Install Windows 11

  • 1
    Download the Media Creation Tool or ISO from Microsoft's official site above.
  • 2
    Plug in a USB drive (8 GB+). Use Rufus or the Media Creation Tool to write the installer to it.
  • 3
    Back up your files to an external drive or cloud. The install will erase your selected drive.
  • 4
    Restart and press F2, F12, DEL, or ESC to enter your BIOS/boot menu. Boot from the USB.
  • 5
    Follow the setup wizard. Choose "Custom: Install Windows only" for a clean install.
  • 6
    Wait 15–30 minutes. Your PC will restart a few times — this is normal. Remove the USB when done!
💡 Tip: If you don't have a product key, you can skip it during install and Windows 11 will still work with minor cosmetic limitations (you can add the key later in Settings).
