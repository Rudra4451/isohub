How to Install Windows 11

  • 1
    Download the Media Creation Tool or ISO from Microsoft's official site above.
  • 2
    Plug in a USB drive (8 GB+) and use the tool or Rufus to flash the installer onto it.
  • 3
    Back up all your files to cloud or external storage — this will erase your drive.
  • 4
    Restart and press F2, F12, DEL, or ESC to enter BIOS. Set USB as first boot device.
  • 5
    Select "Custom: Install Windows only" for a clean install. Follow the wizard.
  • 6
    Wait 15–30 minutes. PC restarts a few times — remove USB when done!
Tip: No product key? Skip it during install — Windows 11 still works with minor cosmetic limitations. Add the key later via Settings → Activation.
